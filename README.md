# ClinicFlow AI Automation

An AI-powered clinic operations system designed to automate appointment scheduling, patient communication, and administrative workflows.

## Overview

ClinicFlow AI Automation is a portfolio project demonstrating how modern AI and no-code automation can streamline clinic operations. The solution combines a Lovable frontend with n8n workflows, Airtable, OpenAI, and Gmail automation to reduce manual work and improve the patient experience.

## Features

- AI-powered patient chatbot
- Automated appointment booking
- Email confirmation and reminders
- Airtable database integration
- OpenAI-powered responses
- Workflow automation with n8n
- Modern responsive frontend built with Lovable

## Tech Stack

- Lovable
- n8n
- Airtable
- OpenAI API
- Gmail
- GitHub

## Use Case

A patient can request an appointment through the chatbot. The workflow validates the request, stores patient information in Airtable, generates an AI-assisted response using OpenAI, and automatically sends a confirmation email through Gmail.
